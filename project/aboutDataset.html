FIFA 18 Complete player dataset

Source :

Kaggle

What does dataset contain ? Information about all the players featured in FIFA 18 (about 17k players)such as:
-> Name, age, photo, nationality
-> Overall rating, potential rating
-> Club, value, wage
-> Attributes : crossing, ball control, passing etc.
-> Positions and prefered postion
Inspiration : What could be we do with raw data and visualize something which would help us to depict some facts which would be harder to depict from the dataset.
Visualizations done so far: Concept map : For top 100 players, used to depict which relationship between club, countries and players
Geo map : To display density of players from each country
Astral plot : To comapre stats between 2 players
Bar chart #1 : To display average salary by position
Bar chart #2 : To display overall and their potential

Scatter plot : For overall, age and wage
